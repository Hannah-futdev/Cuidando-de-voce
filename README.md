# Cuidando-de-voce
Uma ideia de streaming em que o usuário pode escolher realizar diferentes tipos de exercício, assistir vídeos de temas relacionados à saúde e alimentação e também marcar consultas com profissionais da saúde variados.
An idea for a streaming where the user has the chance to do different types of exercises, watch some videos about food and health, and also make appointments  with various health professionals.
